# AGENTS.md: Neuro-Symbolic CoT Pipeline for IDR Binder Design

**Objective:** Build a Python-based, end-to-end differentiable Chain-of-Thought (CoT) pipeline to design high-affinity protein binders for Intrinsically Disordered Regions (IDRs).
**Core Paradigm:** You will build a "Neuro-Symbolic" architecture. A custom PyTorch relaxation engine acts as the symbolic physics/logic evaluator, and extant non-Baker generative models act as the neural proposal generators.

## 3. Module Specifications & Execution Flow

The pipeline consists of three distinct modules that must pass tensors to one another seamlessly.

### Module 1: The Premise (Ensemble Processing)
**Input:** A directory of ~1,000 `.pdb` files containing coarse-grained (C-alpha only) IDR conformations generated by STARLING.
**Tasks:**
1. **C-beta Projection:** Write a PyTorch function to calculate C-beta vectors using the cross-products of the ideal N, C-alpha, and C backbone geometry. Do *not* map to all-atom representation. 
2. **Fractional Contact Map:** Compute a pairwise C-beta to C-beta distance matrix for each conformer. Average these matrices across all 1,000 states to create a probability map. The threshold for interaction is 8 Angstroms.
3. **Motif Extraction:** Implement a sliding window function (window size = 6 to 10 residues) over the contact map. Identify the "Anchor Motif": the continuous sequence with the lowest integrated intra-chain contact frequency and highest sequence hydrophobicity. 
**Output:** The 3D coordinates and sequence of the 6-10 residue Anchor Motif.

### Module 2: The Reasoning Engine (PyTorch CoT Optimization)
**Input:** The Anchor Motif coordinates. 
**Tasks:**
1. **Geometric Initialization:** Place a "Seed" of 3 to 4 interacting dummy amino acids geometrically adjacent to the Anchor Motif, orienting the Seed's C-beta vectors toward the Motif's outward-facing hydrophobic residues.
2. **Continuous Sequence Representation:** Represent the sequence of the Seed not as discrete characters, but as an N x 20 matrix of logits. Implement **Gumbel-Softmax** reparameterization to allow gradients to flow through amino acid selection.
3. **Differentiable Physics Engine:** Build a lightweight loss function (L_total) comprising:
    * An attractive Lennard-Jones/Van der Waals term between the Seed and the Motif.
    * A repulsive steric penalty.
    * An implicit solvent penalty (e.g., exposed hydrophobic surface area).
    * A harmonic tethering constraint strictly applied to the IDR C-alpha atoms (U = k(x - x_0)^2) to allow micro-flexibility without global collapse.
4. **Optimization Loop:** Run gradient descent (Adam optimizer). Backpropagate through the spatial coordinates and the Gumbel-Softmax logits simultaneously to minimize L_total while annealing the temperature parameter (tau -> 0.1). 
**Output:** An optimized 3-4 residue Seed (discrete sequence and 3D coordinates) tightly docked to the Anchor Motif.

### Module 3: Target-Aware Scaffolding & Validation
**Input:** The optimized Seed + IDR Anchor Motif.
**Tasks:**
1. **Steric Shielding Construction:** Take the 6-10 residue Anchor Motif and pad it with 5-10 native residues from the original IDR sequence on both the N- and C-termini. This creates a ~20-residue "Steric Shield."
2. **Scaffolding:** Format the inputs for `Chroma` or `ESM3`. 
    * Define the IDR Steric Shield as fixed context (Sequence + Coordinates fixed).
    * Define the Seed as the anchor (Sequence + Coordinates fixed).
    * Prompt the model to hallucinate a 60-100 residue monomeric scaffold connecting the Seed residues. Apply maximum steric penalties to prevent the scaffold from burying the binding face or entering the volume occupied by the IDR Steric Shield.
3. **Validation:** Pipe the final sequence (Scaffold + IDR) into `Chai-1` via API or local inference. Calculate the predicted RMSD between the Chai-1 folded interface and your PyTorch-designed interface.

## 4. Critical Anti-Patterns (Do NOT do this)
* **The Rotamer Trap:** Do not run side-chain packing (SCWRL, FastRelax) on the IDR ensemble *before* calculating the contact map. Use the C-beta centroid projection to avoid artificial rotamer locking.
* **The Pore Artifact:** Do not pass *only* the 6-10 residue Anchor Motif into the scaffolding model. You must include the flanking regions (the Steric Shield) to force the generator to build a surface cleft rather than a deep tunnel, preserving IDR entropy.
* **Mean Binding Energy:** Do not average binding energies across multiple states if you upgrade to multi-state design. Use a LogSumExp (soft-max) loss to penalize the worst-performing states.
